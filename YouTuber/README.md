## BlogApp YouTuber

This is a customized version of BlogApp made for YouTube channels, also written in 2015, it's a native app that lists videos and playlists using the YouTube API and open the YouTube player inside the app, also receive push notifications of new YouTube videos.

This app was not released on Play Store and probably it's unfinished, as I don't remember much things about these apps because it was written 10 years ago.

## Screenshots

### Android

Adapting the app is very easy (even though no one would do that nowadays because it uses the old Android Java structure), just needing to editing the [strings.xml](./res/values/strings.xml) file and adapting the back-end on the [../Back-end/youtuber](../Back-end/youtuber) directory.
